# Design Docs

Design documents for AgoraFold: architecture decisions, feature designs, and RFCs.

## Contents

- `project-spec.md` — full domain spec and feature scope, forward-looking
- `mvc-razorpages-architecture.md` — architecture notes for the MVC and Razor Pages variants
- `webapi-architecture.md` — architecture notes for the Web API backend and its JS frontend variants (Vue, React, Svelte, Angular, SolidJS)
- `htmx-architecture.md` — architecture notes for the HTMX variant
- `blazor-server-architecture.md` — architecture notes for the Blazor Server variant
- `blazor-wasm-architecture.md` — architecture notes for the Blazor WebAssembly variant
- `backlog.md` — known issues and follow-ups that aren't part of any current task

## Conventions

- Use Mermaid syntax for diagrams.
- Keep this file's Contents list up to date when adding, removing, or renaming files in this folder.
